# StudioTone

Sistema de gerenciamento de loja de instrumentos musicais em PHP + MySQL.

## Funcionalidades
- Login com sessão
- Cadastro de clientes, produtos e categorias
- Registro de vendas
- Relatório de faturamento
- Preenchimento automático de endereço por CEP (ViaCEP)

## Instalação

1. Importe o `script.sql` no MySQL
2. Edite `conexao.php` com suas credenciais
3. Coloque a pasta no `htdocs` e acesse `http://localhost/loja2.0`
